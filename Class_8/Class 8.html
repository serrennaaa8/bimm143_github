Background

The goal of this mini-project is for you to explore a complete analysis using the unsupervised learning techniques covered in class.

The data itself comes from the Wisconsin Breast Cancer Diagnostic Data Set first reported by K. P. Benne and O. L. Mangasarian: “Robust Linear Programming Discrimination of Two Linearly Inseparable Sets”.

Values in this data set describe characteristics of the cell nuclei present in digitized images of a fine needle aspiration (FNA) of a breast mass.

Q12. Can you find a better cluster vs diagnoses match by cutting into a different number of clusters between 2 and 10?

wisc.hclust <- hclust(dist(scale(wisc.df[ , -1]), method = "euclidean"),
                      method = "complete")

diagnosis <- factor(wisc.df$diagnosis)   # B = benign, M = malignant

purity_score <- function(tbl) sum(apply(tbl, 1, max)) / sum(tbl)

best_k <- NA
best_purity <- -Inf

for (k in 2:10) {
  clus   <- cutree(wisc.hclust, k = k)
  tbl    <- table(clus, diagnosis)
  pur    <- purity_score(tbl)
  if (pur > best_purity) {
    best_purity <- pur
    best_k      <- k
  }
}

cat("Best k =", best_k, "\nPurity =", round(best_purity, 3), "\n")
Best k = 9 
Purity = 0.91

6. Sensivity/Specificity

Q17. Which of your analysis procedures resulted in a clustering model with the best specificity? How about sensivity?

The four cluster hierarchical model because it split the entire data into 4 groups and kept almost all of the benign samples together in one clean group. That gave the highest specificity. The two cluster cut grouped almost all the malignant samples together in one big cluster, even if a few benign cases got mixed in. That gave the highest sensitivity.
